# Политика конфиденциальности — WordFlip: Учи испанский

**Дата вступления в силу:** 3 июня 2026 г.

## 1. Общие положения

WordFlip («приложение») разработано Азатом Исмагиловым («разработчик»). Настоящая политика конфиденциальности описывает, какие данные собирает приложение и как они используются.

## 2. Какие данные мы собираем

WordFlip **не собирает и не передаёт** третьим лицам никакие персональные данные пользователей.

Все данные хранятся исключительно на вашем устройстве:
- имя пользователя (задаётся вами при первом запуске)
- прогресс изучения испанских слов
- настройки приложения

Данные не синхронизируются с iCloud и не покидают устройство.

## 3. Сторонние сервисы

Приложение не использует аналитику, рекламные сети, трекеры, SDK третьих сторон или иные сервисы, собирающие данные пользователей.

## 4. Разрешения устройства

Приложение может запрашивать доступ к микрофону и распознаванию речи только для режима тренировки произношения. Распознанный текст используется исключительно внутри текущего упражнения и не сохраняется как персональные данные.

Функция произношения использует стандартный iOS Text-to-Speech (AVSpeechSynthesizer) и работает полностью на устройстве без обращения к серверам.

Приложение может запрашивать разрешение на локальные уведомления, если пользователь включает ежедневные напоминания. Такие уведомления создаются на устройстве и не передают данные разработчику.

## 5. Интернет-соединение

Приложение работает в полностью офлайн-режиме. Интернет-соединение для обучения не требуется и не используется.

## 6. Дети

Приложение не собирает данные о детях и соответствует требованиям COPPA (Children's Online Privacy Protection Act). Приложение безопасно для использования детьми.

## 7. Права пользователей (GDPR)

Поскольку приложение не собирает персональные данные, права на доступ, исправление и удаление данных реализуются путём удаления приложения с устройства. После удаления все локальные данные уничтожаются.

## 8. Изменения политики

В случае изменения настоящей политики обновлённая версия будет опубликована по данному адресу с указанием новой даты вступления в силу.
